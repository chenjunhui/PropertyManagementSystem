# 物业管理系统 — 项目介绍

本目录包含公众号文章原文与配图索引。

| 文件 | 说明 |
|------|------|
| `article.md` | 公众号文章 Markdown 原文 |
| `wechat-post/` | 微信公众号贴图（900px 宽 PNG + 同目录文案 .txt） |
| `wechat-post/post-copy.txt` | 发文标题、摘要与贴图顺序总览 |
| `screenshots.json` | 各模块截图 OSS 地址索引 |
| `pdd-product.json` | 拼多多 1:1 商品主图索引（如有） |
| `pdd-product/` | 拼多多 1:1 商品主图 PNG 文件 |

配图文件托管于 OSS，详见 `screenshots.json` / `pdd-product.json` 中的 `url` 字段。
