# 物业管理系统 — 环境部署说明

> 项目目录：`PropertyManagementSystem` · 技术栈：Spring Boot 3 + Vue 3 + MySQL 8

## 环境要求

| 组件 | 版本 |
|------|------|
| JDK | 17+ |
| Maven | 3.8+ |
| Node.js | 18+ |
| MySQL | 8.x |

## 端口与数据库

| 项 | 值 |
|----|-----|
| 后端端口 | 2010 |
| 管理端端口 | 4010 |
| 客户端端口 | 3010 |
| 数据库名 | `rb_pms` |

## 一、初始化数据库

推荐使用 `02-SQL完整脚本/00_full.sql` 一键导入；或按顺序执行：

1. `01_create_database.sql`
2. `02_schema.sql`
3. `03_mock_data.sql`（可选，仅 Mock 数据见 `03-SQL初期Mock数据/`）

```bash
mysql -u root -p < 00_full.sql
```

## 二、启动服务

### 1. 后端

```bash
cd ../01-项目源码/backend
mvn spring-boot:run
```

验证：http://localhost:2010/api/dashboard/stats

### 2. 管理端

```bash
cd ../01-项目源码/frontend-admin
npm install
npm run dev
```

访问 http://localhost:4010

### 3. 客户端

```bash
cd ../01-项目源码/frontend-client
npm install
npm run dev
```

访问 http://localhost:3010

## 常见问题

- **端口占用**：修改 `backend/src/main/resources/application.yml` 与各前端 `vite.config.js`
- **数据库连接失败**：检查 MySQL 服务、账号密码及库名 `rb_pms`
- **前端依赖安装慢**：可配置 npm 镜像源
