# 物业管理系统 — 项目文档

## 1. 项目概述

| 项 | 内容 |
|----|------|
| 项目名称 | 物业管理系统 |
| 英文名称 | Property Management System |
| 关键词 | 物业、小区、楼栋、房屋、业主、物业费、报修、管理 |
| 技术栈 | Spring Boot 3 + Vue 3 + MySQL 8 |
| 项目类型 | 管理系统 + 客户端 |

## 3. 功能模块

- **数据概览**（admin）：统计楼栋、房屋、业主与报修概况
- **楼栋管理**（admin）：小区楼栋信息维护
- **房屋管理**（admin）：房屋单元与面积管理
- **业主管理**（admin）：业主档案与房屋绑定
- **物业费管理**（admin）：物业费账单生成与缴费状态
- **报修管理**（admin）：报修受理与处理跟踪
- **访客登记**（admin）：访客来访记录与审批
- **公告通知**（admin）：小区通知与物业公告发布
- **巡检记录**（admin）：公共区域巡检与整改记录
- **业主注册**（client）：新业主在线注册账号
- **个人信息**（client）：业主查看与修改个人资料
- **我的房屋**（client）：查看房屋与产权信息
- **报修申请**（client）：在线提交房屋设施报修
- **公告通知**（client）：查看小区通知与物业公告
- **访客预约**（client）：业主为访客提交来访预约

## 4. 角色说明

| 角色 | 说明 | 访问端 |
|------|------|--------|
| 管理员 | 系统全部功能 | 管理端 |
| 物业管家 | 辅助管理，权限可配置 | 管理端 |
| 业主 | 查看房屋、提交报修、预约访客 | 客户端 |

## 5. 数据库设计

- `sys_user` — 系统用户
- `prop_building` — 小区楼栋
- `prop_unit` — 房屋单元
- `owner` — 业主档案
- `property_fee` — 物业费账单
- `repair_request` — 报修申请
- `visitor_record` — 访客登记
- `prop_announcement` — 公告通知
- `inspection_record` — 巡检记录

## 6. 运行说明

### 后端

```bash
cd backend
mvn spring-boot:run
```

### 管理端

```bash
cd frontend-admin
npm install
npm run dev
```

### 客户端

```bash
cd frontend-client
npm install
npm run dev
```
